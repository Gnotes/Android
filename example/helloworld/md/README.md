## build.gradle

build.gradle中包含了我们编译打包时需要的一些信息，如：

- `compileSdkVersion` : 编译SDK版本
- `buildToolsVersion` : 构建工具API版本
- `minSdkVersion` : API最小支持版本
- `targetSdkVersion` : 目标SDK版本
- `versionCode` : 打包apk版本
- `versionName` : 打包apk版本显示名称

具体配置参见[`Gradle`](../../../Gradle/README.md)
