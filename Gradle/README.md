# Gradle

**Gradle** _是一个基于Apache Ant和Apache Maven概念的项目自动化建构工具。它使用一种基于Groovy的特定领域语言来声明项目设置，而不是传统的XML_ -- [Wiki Gradle](https://zh.wikipedia.org/wiki/Gradle)    

简单来说：Gradle就是一个编译脚本（Build System），使用了更简洁的方式书写编译配置。
